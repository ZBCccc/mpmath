# mpMath 测试用例

每个用例的操作方式：点击工具栏「公式」→「插入公式」，将下方 LaTeX 代码粘贴进输入框，观察预览与插入结果。

---

## 一、基础功能

### T01 — 行内公式（inline）

勾选「行间公式」**关闭**，插入以下公式，验证公式与文字在同一行内排列，垂直居中对齐。

```
E = mc^2
```

期望：公式嵌在文字中，不独占一行。

---

### T02 — 行间公式（block）

勾选「行间公式」**开启**，插入以下公式，验证公式独占一行且水平居中。

```
E = mc^2
```

期望：公式单独成行，水平居中，上下有间距。

---

### T03 — 键盘快捷键打开弹窗

在编辑器中按 `Ctrl+/`（Windows）或 `Cmd+/`（macOS），验证弹窗正常弹出。

---

### T04 — Shift+Enter 插入 / Esc 关闭

打开弹窗后输入任意公式，按 `Shift+Enter` 插入；再次打开弹窗，按 `Esc` 关闭，不插入任何内容。

---

## 二、点击已有公式进行编辑（测试竞态修复）

### T05 — 二次编辑单个公式

1. 插入行内公式 `a^2 + b^2 = c^2`
2. 点击已插入的公式，验证弹窗自动打开且输入框已填入原始 LaTeX
3. 将公式改为 `a^2 + b^2 \neq c^2`，点击插入
4. 验证原公式被替换，文章中不出现重复公式

---

### T06 — 连续快速点击同一公式（测试操作锁）

1. 插入一个公式
2. 快速双击该公式
3. 验证弹窗只打开一次，不出现两个弹窗或两次插入

---

### T07 — 连续编辑多个公式

1. 插入公式 A：`x_1`
2. 插入公式 B：`x_2`
3. 点击公式 A 编辑，改为 `y_1`，插入
4. 点击公式 B 编辑，改为 `y_2`，插入
5. 验证两次替换都对应正确的公式位置，没有串位

---

## 三、弹窗交互（测试 focusout 修复）

### T08 — 鼠标点击「插入」按钮

1. 打开弹窗，输入 `\alpha + \beta`
2. **用鼠标点击「插入」按钮**（不使用键盘快捷键）
3. 验证公式正常插入，弹窗关闭

---

### T09 — 鼠标点击「取消」按钮

1. 打开弹窗，输入任意内容
2. **用鼠标点击「取消」按钮**
3. 验证弹窗关闭，编辑器内容不变

---

### T10 — 鼠标点击右上角关闭按钮（×）

同 T09，改用右上角 × 按钮关闭。

---

### T11 — 切换「行间公式」复选框

1. 打开弹窗，输入 `\sum_{i=1}^{n} i`
2. 勾选/取消「行间公式」复选框，观察预览立即更新（不需要重新输入）
3. 验证预览区域正确切换 inline / block 样式

---

## 四、实时预览防抖（测试 debounce 修复）

### T12 — 快速连续输入

在输入框快速打出 `\frac{1}{2} + \frac{1}{3}`，验证：
- 输入过程中预览不会频繁闪烁
- 停止输入约 300ms 后预览更新
- 最终预览与输入内容一致

---

## 五、各类 LaTeX 构造

### T13 — 分数

```
\frac{a+b}{c+d}
```

---

### T14 — 上下标组合

```
x_i^{(k)} = \sum_{j=1}^{n} w_{ij} \cdot x_j^{(k-1)}
```

---

### T15 — 希腊字母

```
\alpha, \beta, \gamma, \delta, \epsilon, \zeta, \eta, \theta, \lambda, \mu, \pi, \sigma, \omega
```

---

### T16 — 求和与积分

行间公式：
```
\sum_{i=0}^{\infty} \frac{x^i}{i!} = e^x
```

行间公式：
```
\int_{-\infty}^{+\infty} e^{-x^2} \, dx = \sqrt{\pi}
```

---

### T17 — 极限

```
\lim_{n \to \infty} \left(1 + \frac{1}{n}\right)^n = e
```

---

### T18 — 矩阵

行间公式：
```
\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax+by \\ cx+dy \end{pmatrix}
```

---

### T19 — 多行对齐方程（aligned）

行间公式：
```
\begin{aligned} f(x) &= (x+1)^2 \\ &= x^2 + 2x + 1 \end{aligned}
```

---

### T20 — 集合与逻辑符号

```
A \cup B, \quad A \cap B, \quad A \subseteq B, \quad \forall x \in \mathbb{R}, \quad \exists \delta > 0
```

---

### T21 — 根号与嵌套

```
\sqrt[3]{\frac{x^2 + \sqrt{x}}{1 + x^4}}
```

---

### T22 — \displaystyle 强制行内公式使用展示样式

行内公式（不勾选「行间」），输入：
```
\displaystyle\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
```

期望：行内公式中求和符号使用完整展示尺寸。

---

### T23 — 带编号的方程（AMS tags）

行间公式：
```
E = mc^2 \tag{1}
```

---

## 六、边缘情况

### T24 — 空输入

打开弹窗，不输入任何内容，验证「插入」按钮处于禁用状态（灰色），无法点击。

---

### T25 — 非法 LaTeX

输入以下内容，验证预览区显示错误信息而不是崩溃，「插入」按钮保持禁用：

```
\frac{}{
```

---

### T26 — 超长公式

行间公式：
```
\frac{d}{dx}\left[\int_{a}^{x} f(t)\,dt\right] = f(x), \quad \forall x \in [a, b], \quad \text{provided } f \text{ is continuous on } [a,b]
```

期望：公式不超出编辑器宽度，`max-width:100%` 样式生效。

---

### T27 — 同段落多个行内公式

在同一段文字中连续插入三个行内公式：`a`、`b`、`c`，验证三个公式均与文字正常对齐，没有多余空格或换行。

---

### T28 — 撤销（Undo）

1. 插入一个公式
2. 按 `Cmd+Z` / `Ctrl+Z` 撤销
3. 验证编辑器行为（注：此处为微信编辑器自身 undo 能力，结果仅供记录，不作为 pass/fail 标准）
